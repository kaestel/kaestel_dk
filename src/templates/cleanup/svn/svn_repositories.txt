New repository:

svnadmin create /projects/svn/repos-name
sudo chown -R daemon /projects/svn/repos-name
sudo chown -R apache /projects/svn/repos-name

sudo chgrp -R daemon /projects/svn/repos-name
sudo chgrp -R apache /projects/svn/repos-name

sudo chmod 777 /projects/svn/repos-name


Import recommended structure:
svn import /projects/template http://localhost/svn/repos-name -m 'initial import'

or from you own computer:
svn import /your-files http://80.165.109.154/svn/repos-name -m 'initial import'


Auto replication:

Checkout replication version for apache if you wish (on server)
svn checkout http://localhost/svn/repos-name/trunk /projects/repos-name/current code

cp /projects/svn/template/hooks/post-commit /projects/svn/repos-name/hooks/post-commit

sudo chown nobody /projects/svn/repos-name/hooks/post-commit
sudo chown -R nobody /projects/repos-name

sudo chown daemon /projects/svn/repos-name/hooks/post-commit
sudo chown -R daemon /projects/repos-name
