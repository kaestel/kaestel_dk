
STASH:
git stash - stashes all changes and reverts repo


ADD TAG:
git tag -a v0.5 -m 'Version 0.5'
git push --tags

DELETE TAG:
git tag -d v0.5
git push origin :v0.5



ADD SUBMODULE:
git submodule add https://github.com/hvadhedderde/repo.git src/build
git commit -m 'added submodule'
git push
git status

CLONE REPO WITH SUBMODULES:
git clone --recursive https://github.com/hvadhedderde/repo.git test


UPDATING REPO WITH NEWLY ADDED SUBMODULE
git submodule init
git submodule update


UPDATE SUBMODULES REPO:
git pull
git submodule update

FORCE ONE SUBMODULE REPO TO LATEST:
cd sub/mod/path
git checkout master
git pull
cd ../parent/repo/path
git add .
git commit -m 'updated sub'
git push


FORCE ALL SUBMODULES REPOS TO LATEST:
git submodule foreach git pull origin master
git add .
git commit -m 'submodule updated'
git push


MODIFY SUBMODULE REPO LOCALLY:
cd sub/mod/path
git checkout master
git pull
<modify code>
git add .
git commit -m 'sub updated'
git push
cd ../parent/repo/path
git add sub/mod/path
git commit -m 'updated sub'
git push


UPDATE SUBMODULE REPO URL:
.gitmodules (correct url)
git submodule sync


DELETE SUBMODULE REPO:
mate .gitmodules
mate .git/config
git rm --cached path/to/submodule

MOVE SUBMODULE REPO:
.gitmodules (delete repo entry)
.git/config (delete repo entry)
git rm --cached path/to/submodule
git commit -m 'deleted submodule'
git push

git submodule add https://github.com/hvadhedderde/repo.git new/path
git commit -m 'added submodule'
git push


MOVE REPO TO OTHER HOST:
git remote rm origin
git remote add origin https://github.com/hvadhedderde/repo.git
git push -u origin master



DELETE ALL TRACES OF FILE
git filter-branch --force --index-filter 'git rm --cached --ignore-unmatch #path/filename#' --prune-empty --tag-name-filter cat -- --all
git push origin master --force
